3x Learning Management Platforms

Variations of a scalable LMS designed and developed for Assesspal, Techie Panda, and Coding School.

The Challenge

Three different brands needed similar core functionality (video hosting, quizzes, progress tracking) but with unique visual identities and branding.

My Role & Process

I designed a "White Label" core architecture. I developed the frontend using a modular approach, allowing me to quickly reskin the application using CSS variables and specific component tweaks for each brand.

The Outcome

Successfully delivered three distinct, fully functional web platforms in a fraction of the time it would take to build them individually.
