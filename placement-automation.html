Placement Automation Dashboard

A unified data dashboard connecting university students to employer placement drives with real-time automated filtering.

Architecture

Built on React using a highly componentized architecture. It utilizes WebSocket connections to stream real-time updates regarding student applications and employer interview statuses. The global state is managed tightly to ensure grid re-renders only occur when explicitly required.

Challenges Solved

  • Main Thread Blocking: Extracting nested JSON data for 10,000+ student records caused the browser UI to freeze. Shifted data parsing and heavy sorting algorithms to an asynchronous Web Worker.
  • DOM Node Exceedance: Rendering thousands of table rows natively crashed mobile browsers. Replaced native tables with a windowed approach using React Virtualized.

Performance Improvements

By implementing purely virtualized scrolling, the DOM complexity dropped from 250,000+ nodes to just 150. This slashed rendering time from 4.2 seconds down to ~200ms, providing a flawlessly smooth 60fps scrolling experience regardless of dataset size.
