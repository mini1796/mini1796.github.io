TopicPrep Admin Web App

The comprehensive admin dashboard used to manage content and users for the mobile app.

The Problem

The content team was struggling to upload questions and manage student data using scattered spreadsheets. They needed a centralized, user-friendly interface to manage the TopicPrep ecosystem.

My Role & Process

I designed a clean, data-heavy web interface focusing on efficiency. Key features included bulk-upload interfaces for questions, student performance analytics charts, and user management tables.

The Outcome

The admin panel streamlined operations, allowing the content team to publish quizzes 50% faster than the previous manual method.
