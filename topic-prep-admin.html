TopicPrep Admin Web App

The comprehensive admin dashboard used to manage content and users for the mobile app.

Problem

The content team was struggling to upload questions and manage student data using scattered spreadsheets. They needed a centralized, user-friendly interface to manage the TopicPrep ecosystem and quickly process massive bulks of data.

User Flow

Admin Logs in → Dashboard shows daily high-level KPIs → Admin navigates to Content Manager → Uses Bulk-Upload tool to ingest a CSV → System flags errors → Content is published to mobile app instantly.

Screens

Screens highlight data-heavy web interfaces designed for efficiency. Key features include bulk-upload drag-and-drops, student performance analytics charts, and sticky user management tables.
